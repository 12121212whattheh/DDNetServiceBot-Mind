# How to start

**First of all**: `yarn tsc`

## Start from koishi

```
yarn koishi start ./dist/koishi.config.js
```

## Start from nodejs

```
node ./dist/index.js
```

## Start from pm2

```
// pm2 start ./dist/index.js --name koishi --log ./mot.log
pm2 start pm2.json
```
